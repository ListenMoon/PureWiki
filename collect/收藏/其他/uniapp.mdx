### **[image-tools](https://ext.dcloud.net.cn/plugin?id=123)**

图像转换工具，可用于图像和base64的转换

可配合uni.saveImageToPhotosAlbum保存相册

```
                base64ToPath(this.qrcode)
                  .then(path => {
                    uni.saveImageToPhotosAlbum({
                        filePath: path,
                        success: function() {
                            that.$toast('保存成功');
                            that.$hideLoading()
                        },
                        fail: err => {
                            that.$toast('保存失败');
                            that.$hideLoading()
                        }
                    });
                  })
                  .catch(error => {
                    console.error(error)
                  })
```
