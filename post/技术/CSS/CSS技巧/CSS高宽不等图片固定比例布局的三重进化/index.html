aspect-ratio 属性

到了 2021 年，随着 CSS 技术的发展，固定比例的图片布局有了新的选择，这个就是 CSS aspect-ratio属性，专门设置元素的高宽比。

不过此属性用在 IMG 元素上是没有效果的，因为图片元素具有内在尺寸，不受 aspect-ratio 属性影响。

因此，使用 aspect-ratio 属性实现等比例图像弹性布局需要至少两层标签。

比 padding 方法，CSS 自然也简化了些，这里使用 Flex 布局示意。

是不是容易理解多了。

实现效果类似下面的:

demo
查看源码

<style>
    .box {
        list-style: none;
        display: flex;
        gap: 0.5rem;
        flex-wrap: wrap;
    }
    .list {
        flex-basis: calc(25% - 1.5rem / 4);
        aspect-ratio: 1 / 1;
    }
    .list img {
        display: block;
        width: 100%;
        height: 100%;
        object-fit: cover;
    }
</style>
<div style="resize: horizontal;overflow: hidden;">
    <ul class="box">

cqw 单位

到了如今的 2023 年，又有了更加简单的方法，实现最终的布局需要只需要一层容器标签即可。

这么给力，那是用什么东西实现的呢？

嘿嘿，容器元素和 cqw 单位，具体 CSS 如下所示:

container-type:inline-size可以让普通元素变成 container 容器元素，而 cqw 是容器宽度单位，1cqw=1%的容器宽度，100cqw 就是容器宽度。

而本需求需要一行显示 4 个图片，因此，每个图片相对于容器的大小就很好计算了。

就是这么简单。

效果一样：

demo
查看源码

<style>
    .box {
        list-style: none;
        display: flex;
        gap: 0.5rem;
        flex-wrap: wrap;
        container-type: inline-size;
    }
    .box img {
        width: calc(25cqw - 1.5rem / 4);
        height: calc(25cqw - 1.5rem / 4);
        object-fit: cover;
    }
</style>
<div style="resize: horizontal;overflow: hidden;">
    <div class="box">
        <img src="/avatar.jpg" />
        <img src="/avatar.jpg" />
        <img src="/avatar.jpg" />
        <img src="/avatar.jpg" />
        <img src="/avatar.jpg" />
        <img src="/avatar.jpg" />
    </div>
</div>
