## Github统计卡片

```
<img align="right" src="https://github-readme-stats.vercel.app/api?username=npmrun&show_icons=true&icon_color=CE1D2D&text_color=718096&bg_color=ffffff&hide_title=true" />
```

只需要把 username 改成自己的 GitHub 用户名就可以了。

当然也可以查看[这个项目](https://github.com/anuraghazra/github-readme-stats/blob/master/docs/readme_cn.md)的[官方文档](https://github.com/anuraghazra/github-readme-stats/blob/master/docs/readme_cn.md)，进行个性化设置，比如选择[主题](https://github.com/anuraghazra/github-readme-stats/blob/master/themes/README.md)，或者自己选择相应板块颜色。
