:collect[OverlayScrollbars||一个漂亮的滚动条插件]{url="https://kingsora.github.io/OverlayScrollbars/#!overview||官网"} 是一个JavaScript滚动条插件，隐藏本机滚动条，提供自定义的可样式覆盖滚动条，并保持本机功能和感觉。

:::demo
```html
<script src="
https://cdn.jsdelivr.net/npm/overlayscrollbars@2.1.0/browser/overlayscrollbars.browser.es6.min.js
"></script>
<link href="
https://cdn.jsdelivr.net/npm/overlayscrollbars@2.1.0/styles/overlayscrollbars.min.css
" rel="stylesheet">
<div class="sc1">
    <div>1111</div>
    <div>1111</div>
    <div>1111</div>
    <div class="sc2">
        <div class="green">1111</div>
    </div>
    <div>1111</div>
    <div>1111</div>
    <div>1111</div>
    <div>1111</div>
</div>
<style>
    html,body{
        margin: 0;
        padding: 0;
        height: 100%;
    }
    .sc1{
        width: 300px;
        height:800px;
        background-color: red;
    }
    .sc2{
        width: 300px;
        height:150px;
        background-color: green;
    }
    .green{
        height:600px;
    }
</style>
<script>
var OverlayScrollbars = OverlayScrollbarsGlobal.OverlayScrollbars;
const defaultOptions = {
    scrollbars: {
        autoHide: 'move', // 鼠标移动时就显示隐藏
    }
}
OverlayScrollbars(document.body, defaultOptions);
OverlayScrollbars(document.querySelector(".sc2"), defaultOptions);
</script>
```
:::
可以看到一个简单的嵌套示例，你可以实例化一个需要滚动的元素，然后传入配置选项，**配置一定要传**，不然没有效果。

## 选项

`OverlayScrollbars`第二个参数为选项配置。其默认选项如下：
```
const defaultOptions = {
  paddingAbsolute: false,
  showNativeOverlaidScrollbars: false,
  update: {
    elementEvents: [['img', 'load']],
    debounce: [0, 33],
    attributes: null,
    ignoreMutation: null,
  },
  overflow: {
    x: 'scroll',
    y: 'scroll',
  },
  scrollbars: {
    theme: 'os-theme-dark',
    visibility: 'auto',
    autoHide: 'never',
    autoHideDelay: 1300,
    dragScroll: true,
    clickScroll: false,
    pointers: ['mouse', 'touch', 'pen'],
  },
};
```
|key|类型|默认值|描述|
|:--:|:--:|:--:|:--:|
|`paddingAbsolute`|`boolean`|`false`| 指示内容的`padding`是否应为绝对值 |
|`showNativeOverlaidScrollbars`|`boolean`| `false`|原生滚动条是否应该显示 |
|`update.elementEvents`|`boolean`|`false`| :popup[详情::这里面是个元组,元组的第一个值是一个选择器和事件名组成的数组，如此，当指定的元素触发了指定的事件，布局就会更新] |
