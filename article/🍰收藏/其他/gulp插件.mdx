## 安装

### 全局

```bash
npm install gulp-cli -g
npm install gulp -D
```

### 局部

```bash
npm install --save-dev gulp
```

### `gulpfile.js`文件或者`gulpfile.js\index.js`

```
function defaultTask(cb) {
  // place code for your default task here
  cb();
}

exports.default = defaultTask
```

### 插件推荐

* `gulp-if`
* `gulp-babel`
* `gulp-plumber`
* `gulp-sourcemaps`
* `gulp-postcss`和`postcss`
* `gulp-data`和`gulp-ejs`
* `gulp-typescript`
* `gulp-filter`
* `gulp-htmlmin`
* `gulp-uglify`
* `del`
* `gulp-replace`

* `cross-env`
